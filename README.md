# Mini site pour planifier un date

Site statique prêt pour GitHub Pages.

## Parcours

1. Page d'introduction avec le bouton `non` impossible.
2. Choix de la date et de l'heure.
3. Choix de l'activité.
4. Récapitulatif à screenshot.

## Personnalisation

- Le texte se modifie dans `index.html`.
- Le style visuel se modifie dans `styles.css`.
- Les comportements se modifient dans `script.js`.
